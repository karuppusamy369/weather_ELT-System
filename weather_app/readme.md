# Weather ETL Pipeline

**Automated Weather ETL Pipeline using Python and MySQL with Analytics and Visualization**

---

## **Project Overview**

This project collects live weather data from the OpenWeatherMap API, stores it in a MySQL database, and visualizes trends over time. The ETL pipeline is **automated to run every 30 minutes**, providing historical weather data for multiple cities.

**Features:**
- Fetch live weather data (temperature, humidity, weather description)
- Store historical data in MySQL
- Automated pipeline using Python (`schedule` library)
- Analytics and visualization of temperature and humidity trends
- Professional, resume-ready structure
